MulticolorEngineRequest..::..CountMetadataCollection Method
MulticolorEngineRequest Class See Also 
Given one of more metadata queries, get a counter for each query specifying how many of the collection images match the query. The images counted in the results may be filtered using json metadata.

Namespace: TinEye.Services
Assembly: TinEyeServices (in TinEyeServices.dll) Version: 1.2.3.0 (1.2.3.0)

Parameters

countMetadata
Type: array<JObject>[]()[][]
Array of metadata queries to get image counts for.
metadata
Type: JObject
Metadata to filter the images counted. Set it to null if it is not used.

Return Value

The MulticolorEngine API JSON response with the count of images matching each metadata query passed in.

Remarks

Returns the MulticolorEngine API JSON response with the following fields:

statusOne of ok, warn or fail
methodcount_metadata
resultArray with a single JSON object which has a metadata field set to a list of JSON objects with the counts for each metadata query passed in, along with the original metadata passed in
errorArray of error messages if status is not ok
