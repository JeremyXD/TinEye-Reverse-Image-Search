Return Value

The MulticolorEngine API JSON response with the count of images matching the colors and each metadata query passed in.

Remarks

A weight may be given for each color indicating how much of a color should appear in a matching image. Weights must be given for each color if weights are included, and each weight must be between 1 and 100 and they all must add to 100.

Returns the MulticolorEngine API JSON response with the following fields:

statusOne of ok, warn or fail
methodcount_metadata
resultArray with a single JSON object which has a metadata field set to a list of JSON objects with the counts for each image with matching colors, for each metadata query passed in.
errorArray of error messages if status is not ok
