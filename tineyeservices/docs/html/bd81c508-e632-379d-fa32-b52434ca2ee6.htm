Type: System..::..Boolean
If true, ignore the background color of the images. If false, include the background color of the images.
ignoreInteriorBackground
Type: System..::..Boolean
If true, ignore regions that have the same color as the background region but that are surrounded by non-background regions.

Return Value

The MulticolorEngine API JSON response with the count of palette colors in the passed in images.

Remarks

If the ignoreBackground option is set to true, an image region containing 75% or more of the image's edge pixels is considered a background region and its associated color is not counted. This is useful if you have an image of objects on a solid background color that you don't want to have counted.

Returns the MulticolorEngine API JSON response with the following fields:

statusOne of ok, warn or fail
methodcount_image_colors
resultArray of JSON objects with the following fields:
colorA color in hex format, or an array with that color's 3 RGB values
num_images_partial_areaThe number of input images that contain the palette color in any percentage
num_images_full_areaThe number of input images that contain the palette color in a very large percentage
errorArray of error messages if status is not ok
