The Image class is used to represent images corresponding to API collection images.

public class Image

Remarks

There are two types of images represented: those to be uploaded using the image data and those that have an image URL but no local image data.

Examples

CopyC#
// Make an Image based on a local image:
Image localImageToUpload = new Image(@"C:\some\dir\with\myImage.jpg");

// Make an Image based on a image with a given URL:
Uri imageUrl = new Uri("http://www.mywebsite.com/images/myImage.jpg", 
                       "/imagepath/in/collection/image.jpg");

Image imageFromUrl = new Image(imageUrl);

For images to be used by the MulticolorEngine API, images can be created with JSON keyword metadata.
